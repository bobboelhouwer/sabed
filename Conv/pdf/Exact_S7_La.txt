Nomenclatuur
Kwadratische vorm Q(x) is:
• Positief definiet: Q(x) > 0 ∀ x≠0, alle λi > 0
• Negatief definiet: Q(x) < 0 ∀ x ≠0, alle λi < 0
• Positief semi-definiet: Q(x) ≥ 0 ∀ x, alle λi ≥ 0
• Negatief semi-definiet: Q(x) ≤ 0 ∀ x, alle λi ≤ 0
• Indefiniet anders, positieve en negatieve λi
• Hetzelfde wordt gezegd van A (symmetrisch)
